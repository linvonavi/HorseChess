# HorseChess
* set board:
  ```set_position <fen>```
* get possible moves:
```get_moves```
* get possible moves from position in algebraic notation:
```get_moves_from_position <position>```
example: ```get_moves_from_position a2```
* make move
```make_move <move>```
example: ```make_move a2a4```
* turn on/off debug
```debug```
